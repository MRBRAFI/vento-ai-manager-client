# VENTO - AI Model Inventory Management System

**Live Site:** https://vento-ai-inventory.web.app/

VENTO is a personal AI model inventory management system designed to help users manage, view, and update AI models efficiently.

## Features

- **User Authentication:** Register, login, and sign in with Google, with profile information including name and photo.
- **Model Management:** Add new AI models, update existing models, and view all models.
- **Private Routes:** Secure pages for adding, updating, and viewing personal models.
- **Responsive Design:** Fully responsive and interactive UI built with React, TailwindCSS, and DaisyUI.
- **Dark/Light Mode Toggle:** Switch between dark and light themes dynamically from the navbar.
- **Loader Integration:** Smooth loading indicators while fetching data or logging in to prevent flickering.
- **Purchase Tracking:** View models you have purchased and manage them easily from your dashboard.

## Tech Stack

- React.js (with Hooks and Context API)
- React Router v6
- TailwindCSS + DaisyUI for styling
- Firebase Authentication (Email/Password + Google Sign-In)
- SweetAlert2 for confirmation dialogs
- React-Toastify for notifications
